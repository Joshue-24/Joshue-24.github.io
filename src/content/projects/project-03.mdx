## Visión General del Proyecto

Este proyecto aborda un aspecto crítico de la seguridad laboral: la supervisión del uso adecuado del Equipo de Protección Personal (EPP). Desarrollamos un sistema de visión por computadora capaz de detectar automáticamente si los trabajadores están utilizando el EPP requerido (como cascos, chalecos reflectantes, guantes, etc.) en tiempo real. El objetivo es prevenir accidentes y asegurar el cumplimiento de las normativas de seguridad en entornos como obras de construcción, fábricas o almacenes, reduciendo la necesidad de supervisión humana constante.

## Implementación Técnica y Tecnologías Clave

La base de este sistema es la detección de objetos de alta velocidad y la clasificación de imágenes.

### Detección de Objetos con YOLO (You Only Look Once)

Utilizamos el algoritmo YOLO (You Only Look Once) por su capacidad para realizar detecciones en tiempo real con alta precisión. Se entrenó un modelo YOLO personalizado con un conjunto de datos que incluye imágenes de trabajadores con y sin diferentes tipos de EPP.

```python
from ultralytics import YOLO
import cv2

# Cargar el modelo YOLOv8 entrenado para EPP
model = YOLO('./models/yolov8n_epp.pt') # Ruta a tu modelo entrenado

# Clases de EPP que el modelo puede detectar
# Asegúrate de que estas coincidan con las clases en tu entrenamiento
class_names = ["casco", "chaleco", "guantes", "gafas"]

# Inicializar la captura de video
cap = cv2.VideoCapture(0) # 0 para la cámara web, o ruta a un archivo de video

while True:
    ret, frame = cap.read()
    if not ret:
        break

    # Realizar inferencia
    results = model(frame)

    # Procesar resultados y dibujar bounding boxes
    for r in results:
        boxes = r.boxes
        for box in boxes:
            x1, y1, x2, y2 = map(int, box.xyxy[0])
            conf = round(float(box.conf[0]), 2)
            cls = int(box.cls[0])
            label = f"{class_names[cls]}: {conf}"

            # Dibujar rectángulo y etiqueta
            cv2.rectangle(frame, (x1, y1), (x2, y2), (0, 255, 0), 2)
            cv2.putText(frame, label, (x1, y1 - 10), cv2.FONT_HERSHEY_SIMPLEX, 0.5, (0, 255, 0), 2)

    cv2.imshow("Detección de EPP", frame)

    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

cap.release()
cv2.destroyAllWindows()
```

### Lógica de Cumplimiento y Alertas

Además de la detección, el sistema incluye una lógica para determinar el cumplimiento del EPP. Por ejemplo, si se detecta un trabajador en una zona específica sin casco, se puede generar una alerta.

```python
# Pseudocódigo para la lógica de cumplimiento
def check_epp_compliance(detections, required_epp_for_zone):
    detected_epp = set()
    for det in detections:
        detected_epp.add(class_names[int(det.cls[0])])

    missing_epp = required_epp_for_zone - detected_epp

    if missing_epp:
        print(f"ALERTA: EPP faltante: {missing_epp}")
        # Aquí se podría activar una alarma, enviar una notificación, etc.
        return False
    else:
        print("Cumplimiento de EPP OK.")
        return True
```
